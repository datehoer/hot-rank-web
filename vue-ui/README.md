# vue-ui

Current frontend for **https://www.hotday.uk/**.

## Stack

- Vue 3
- Vite
- TypeScript
- Tailwind CSS
- Vue Router
- vue-i18n

## Requirements

- Node.js 22.x
- pnpm

## Install

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

Build output will be generated in `dist/`.

## Preview build locally

```bash
pnpm run preview
```

## Lint

```bash
pnpm run lint
```
